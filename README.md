# Overview

## My Cast

A java web app that fetches movies information The Movie Database, and friendly displays top-rated movies to the user.
This web app was built intended for building skills on Spring Boot Framework, applying the pattern MVC.

## How To Run

-> All dependencies are already configured. However, you still need to add a Firebase API key file into the root directory of resources. the application properties is expecting this key to be named as: "mycast-app-firebase.json".

-> After you added the API key file, run the class MycastApplication inside "src/main/java/com.mycast/"

## Development Environment

- Chosen IDE: IntelliJ Community Edition
- Java 25
- Spring Boot 3.5.7
- Thymeleaf (Server-side templates)
- Bootstrap
- GCP Firestore

## Dependencies

- Spring Boot
- Spring Boot Thymeleaf
- Thymeleaf Layout Dialect
- Spring Boot validator
- Spring Security Core (using Bcrypt only)
- GCP Firebase Admin
